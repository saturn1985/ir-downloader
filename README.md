# 📦 IR-Downloader

یک ابزار قدرتمند مبتنی بر GitHub Actions برای دانلود فایل‌ها از لینک مستقیم، دسته‌بندی خودکار، رمزگذاری و ذخیره‌سازی دائمی در مخزن گیت‌هاب. دیگر نگران حذف خودکار فایل‌ها یا محدودیت حجم نباشید.

---

## ✨ قابلیت‌های کلیدی

- 🔗 دانلود از هر لینک مستقیم (HTTP/HTTPS)
- 🧠 تشخیص خودکار نوع فایل → ویدئو / صدا / تصویر / سند / سایر
- 📁 ساختار هوشمند پوشه‌ها
- 🔐 رمزگذاری دلخواه (اختیاری) با 7-Zip
- ✂️ تقسیم به تکه‌های دلخواه (پیش‌فرض 95 مگابایت)
- 💾 ذخیره دائمی در خود مخزن (نه Artifact موقتی)
- 📥 دانلود یکجا با کلیک روی Code → Download ZIP
- 🔄 عدم تداخل فایل‌های قبلی – هر فایل در پوشه جداگانه
- ⏱️ بدون محدودیت زمانی (برخلاف Artifactهای 90 روزه)

⚙️ مراحل گام به گام (فقط چند دقیقه)

1_ پروژه آماده را فورک (کپی) کنید:

•به لینک زیر بروید

🔗https://github.com/ali25az/ir-downloader

• روی دکمه Fork (بالای راست صفحه) کلیک کنید.
• در صفحه جدید، روی Create Fork بزنید.
• چند ثانیه صبر کنید تا یک کپی از پروژه در اکانت شما ساخته شود.

2_ وارد اکانت فورک شده خود شوید
• بعد از فورک، خودکار به صفحه پروژه کپی شده هدایت می‌شوید.

3_ اجرای عملیات دانلود
• روی تب Actions کلیک کنید.
• در سمت چپ، روی «حرفه‌ای: دانلود و ذخیره در مخزن» کلیک کنید.
• دکمه آبی Run workflow را بزنید.
• یک کادر باز می‌شود. در آن:
•لینک مستقیم فایل مورد نظر را وارد کنید.
•(در صورت تمایل) رمز فایل فشرده را وارد کنید.
•سپس روی دکمه سبز Run workflow کلیک کنید و فرآیند دانلود شروع شد.

4_ چگونه بفهمیم دانلود تمام شده است؟
• بسته به حجم فایل، ممکن است چند دقیقه تا چند ساعت طول بکشد.
•برای پیگیری پیشرفت، دوباره به تب Actions بروید.
•روی همان اجرا (run) کلیک کنید.
•روی جعبه download-and-split کلیک کنید تا جزئیات و زمان تقریبی را ببینید.
•وقتی کار آن تمام شد، کنار نام run یک دایره سبز ظاهر می‌شود.
    (نیازی نیست صفحه را باز نگه دارید، می‌توانید بعداً برگردید.)

5_ دریافت فایل دانلود شده:
•بالای صفحه، روی نام مخزن اصلی (مثلاً github-file-downloader) کلیک کنید.
•روی دکمه سبز Code کلیک کنید.
•گزینه Download ZIP را بزنید.
•فایل ZIP دانلود شده را از حالت فشرده خارج کنید.
•داخل آن، یک فایل با پسوند 001. وجود دارد.
•با نرم‌افزارهای استخراج مثل ZArchiver یا WinRAR آن را باز کنید، این نرم‌افزارها به‌طور خودکار تمام تکه‌ها را به هم می‌چسبانند و فایل نهایی با اسم download_file بهتون میده که باید بسته به پسوند فایل اصلیتون براش .mp4 ، .mp3 ، .png و .. بزارید و تمام فایل قابل استفاده شما آماده می‌شود.
